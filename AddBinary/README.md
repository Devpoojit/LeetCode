#   [Add Binary](https://leetcode.com/problems/add-binary/description/)

-   ### Problem Statement:
    -   Given two binary strings a and b, return their sum as a binary string.

-   ### Test Cases:
    -   Case 1:
        Input: a = "11", b = "1"
        Output: "100"

    -   Case 2:
        Input: a = "1010", b = "1011"
        Output: "10101"
